Aula 5 | Intervalos Musicais

Curso 2

Tudo o Que Você Precisa Saber Sobre Intervalos Musicais Se você está começando seus estudos de harmonia musical, entender os intervalos musicais é essencial. Os intervalos são, basicamente, as distâncias entre as notas e são fundamentais para compreender a estrutura da música. Neste artigo, vamos explorar o que são os intervalos, como identificá-los e como utilizá-los na música. O Que São Intervalos Musicais? Em termos simples, intervalos musicais são a distância entre duas notas. Eles podem ser pequenos ou grandes, e são classificados de diferentes maneiras, dependendo da relação entre as notas. O conceito de intervalo é importante porque ele define a sonoridade e a sensação que a música transmite. Existem 12 sons diferentes dentro de uma oitava, e cada um desses sons é uma nota musical. Os intervalos, então, são as distâncias entre essas notas. Vamos entender melhor como isso funciona. O Papel dos Intervalos na Música Os intervalos desempenham um papel importante na música porque são responsáveis pela construção de melodias e harmonias. Embora as notas possam ser diferentes, se as distâncias entre elas forem as mesmas, a sensação auditiva será a mesma. Por exemplo, se você tocar a mesma sequência de notas em diferentes tons, mas mantendo os intervalos constantes, a música soará a mesma, independentemente do tom em que você tocar. Esse conceito é a base de como transpor músicas para diferentes tonalidades. Classificação dos Intervalos Musicais Os intervalos podem ser classificados de várias maneiras. A maneira mais comum de classificá-los é como maiores, menores, justos, aumentados ou diminutos. Vamos ver como cada tipo é formado, utilizando como base a escala de dó maior, que é uma das escalas mais simples de se entender. Escala de Dó Maior A escala de dó maior é composta pelas notas: dó, ré, mi, fá, sol, lá, si. A distância entre as notas dessa escala é sempre a mesma, e segue o padrão: • Tom (dó para ré) • Tom (ré para mi) • Semitom (mi para fá) • Tom (fá para sol) • Tom (sol para lá) • Tom (lá para si) • Semitom (si para dó) A escala de dó maior é uma boa referência porque nela, os intervalos podem ser classificados como maiores ou justos. Intervalos Maiores e Justos 1 Intervalos Maiores: ◦ A segunda (dó para ré) ◦ A terça (dó para mi) ◦ A sexta (dó para lá) ◦ A sétima (dó para si) 2 Intervalos Justos: ◦ A primeira (dó para dó) ◦ A quarta (dó para fá) ◦ A quinta (dó para sol) ◦ A oitava (dó para a próxima nota dó) Como Identificar Intervalos Menores, Aumentados e Diminutos Além dos intervalos maiores e justos, temos os menores, aumentados e diminutos, que são variações dessas distâncias. • Intervalo Maior + 1 Semitom = Aumentado • Intervalo Maior - 1 Semitom = Menor Por exemplo: • De dó para ré temos uma segunda maior. Mas, se tocar de dó para ré bemol, temos uma segunda menor. • Se você tocar de dó para ré sustenido, terá uma segunda aumentada. Os intervalos justos seguem a mesma lógica: • Intervalo Justo + 1 Semitom = Aumentado • Intervalo Justo - 1 Semitom = Diminuto Por exemplo: • De dó para sol temos uma quinta justa. Mas, de dó para sol bemol, temos uma quinta diminuta. • Se tocar de dó para sol sustenido, você terá uma quinta aumentada. Como Inverter Intervalos Um conceito importante é o de inversão de intervalos. Inverter um intervalo significa trocar a ordem das notas. Ou seja, ao invés de tocar de dó para mi, você toca de mi para dó. Quando um intervalo é invertido, a soma das distâncias dos dois intervalos sempre dá 9. Veja como funciona: • Se de dó para mi temos uma terça maior, ao inverter o intervalo e tocar de mi para dó, temos uma sexta menor (3 + 6 = 9). • Da mesma forma, de dó para sol temos uma quinta justa, mas ao inverter, de sol para dó, temos uma quarta justa (5 + 4 = 9). Regras Importantes ao Inverter Intervalos 1 Intervalos Maiores viram Menores quando invertidos e vice-versa. 2 Intervalos Justos continuam Justos quando invertidos e vice-versa. 3 Intervalos Menores viram Aumentados quando invertidos, e Aumentados viram Menores. 4 Intervalos Diminutos viram Aumentados e vice-versa. Como Usar Intervalos na Prática Agora que você sabe o que são os intervalos e como classificá-los, é hora de aplicar esse conhecimento na prática. Os intervalos são a base da construção de melodias, harmonias e acordes. Ao conhecer bem os intervalos, você pode: • Transpor músicas para diferentes tonalidades mantendo a estrutura harmônica. • Criar acordes com intervalos específicos entre as notas. • Compor melodias que soem agradáveis e harmônicas. Conclusão Compreender os intervalos musicais é uma habilidade essencial para quem quer dominar a harmonia e a teoria musical. Eles são a base para entender como as notas se relacionam e como a música é construída. Esperamos que este artigo, junto com a videoaula complementar, tenha esclarecido esse conceito fundamental. Continue praticando e explorando as possibilidades dos intervalos na música, e você verá como eles podem transformar sua forma de entender e criar música!
